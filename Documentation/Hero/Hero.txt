# Hero Component Documentation

## Overview

The `Hero` component is a full-screen landing section featuring a video background with overlay text and a call-to-action button. It serves as the primary visual introduction to the Oakworth platform.

## Features

- **Full-screen video background** with auto-play and looping
- **Responsive typography** that scales across all device sizes
- **Gradient overlay** for improved text readability
- **Scroll indicator** for user guidance
- **Mobile-optimized** viewport handling
- **Accessible** video fallback content

## Props

This component does not accept any props. All content and behavior are hardcoded.

## Video Specifications

### File Requirements
- **Primary Format**: MP4 (H.264 codec)
- **Fallback Format**: WebM (optional)
- **Location**: `/public/1.mp4`
- **Dimensions**: 1920x1080 (1080p) or higher
- **Duration**: 15-30 seconds recommended for smooth looping
- **File Size**: Optimized for web (2-5MB recommended)

### Video Attributes
- `autoPlay`: Starts automatically
- `muted`: Required for autoplay in modern browsers
- `loop`: Continuous playback
- `playsInline`: Prevents full-screen on iOS
- `preload="auto"`: Optimizes loading performance

## Content Structure

### Main Heading
- **Text**: "OAKWORTH"
- **Font**: Serif font family
- **Weight**: Light (300)
- **Color**: White (#FFFFFF)
- **Tracking**: Tight letter spacing

### Subheading
- **Text**: "Definitive LBO intelligence for private equity."
- **Color**: Light gray (#E5E7EB)
- **Max Width**: 2xl (42rem)

### Call-to-Action Button
- **Text**: "Get Started"
- **Background Color**: #0A1F36 (Oakworth brand blue)
- **Border**: None (border-0)
- **Shape**: Square corners (rounded-none)
- **Hover Effects**: 90% opacity with slight scale reduction
- **Link Target**: "#contact" section
